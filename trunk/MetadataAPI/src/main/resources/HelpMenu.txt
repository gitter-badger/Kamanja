'[]' - means that this parameter is optional

I. Message operations
kamanja add message [fileLocation]
kamanja update message [fileLocation]
kamanja get message [fileLocation]
kamanja get all messages
kamanja remove message [fileLocation]

II. Model operations
kamanja add model kpmml [fileLocation]
kamanja add model pmml input MODELNAME namespace.name MODELVERSION nn.nn.nn MESSAGENAME namespace.name
kamanja add model java [fileLocation]
kamanja add model scala [fileLocation]
kamanja get model [fileLocation]
kamanja get all models
kamanja get active models
kamanja remove model [fileLocation]
kamanja update model kpmml [fileLocation]
kamanja update model pmml input MODELNAME namespace.name MODELVERSION nn.nn.nn
kamanja update model scala [fileLocation]
kamanja update model java [fileLocation]
kamanja deactivate model [fileLocation]
kamanja activate model [fileLocation]
